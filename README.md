# Distractor: Copenhagen

One random public-domain work from [SMK — Statens Museum for Kunst](https://www.smk.dk/en/article/smk-api/),
the National Gallery of Denmark. A sibling of Distractor: MoMA (`main`
branch); same client, different collection.

**Data is built in CI:** `api.smk.dk` is not reachable from every sandbox,
so `public/data/` on this branch is produced by the `build-distractors`
GitHub Actions workflow (or run `npm run build` anywhere with open egress).

## Legal

SMK dedicates reproductions of public-domain works to CC0; only records
flagged `public_domain` with an image are included.
